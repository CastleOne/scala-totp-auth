TOTPSecret

class TOTPSecret extends BigInt

The secret is represented as a BigInt. BigInts make it easy to convert between Hex and Base32. Method toByteArray lets you interface with standard crypto. Base32 is the defacto format used by Google authenticator. The OAuthTool uses hex :(
